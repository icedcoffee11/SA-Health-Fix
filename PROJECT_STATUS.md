# Project Status

This repository contains a draft public-policy concept for South Australian health-system flow and ambulance ramping reform.

## Current state

- Status: working draft
- Purpose: policy concept, reform architecture, and communication scaffold
- Main document: `README.md`
- Author/concept lead: Scott Muller

## Verification notes

The proposal includes operational targets, cost ranges, and system-design claims. Before using this material in a formal submission, public campaign, media pitch, or parliamentary context, the following should be checked against current primary sources:

1. Latest SA Health ramping figures.
2. Latest ambulance offload performance figures.
3. Current number of medically fit patients awaiting aged care, NDIS, rehab, housing, or other discharge pathways.
4. Updated acute bed-day cost assumptions.
5. Current SA Health budget figures.
6. Commonwealth funding pathways and intergovernmental responsibilities.
7. Procurement, privacy, clinical safety, and Indigenous data-governance requirements for any AI component.
